# Deepfake Detection Project

## Overview
This project is a Deepfake Detection system built using Deep Learning techniques.  
It classifies whether an image/video frame is real or fake.


## Model
- PyTorch-based CNN model
- Trained using custom dataset
- Saved model weights: `best_model_state_dict.pth` (stored externally)

## Project Structure
- notebooks/ → training and experiments
- src/ → model and utility code
- dataset/ → data processing (if included)
